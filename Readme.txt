OVERVIEW
--------
This game was built in two hours as an example of simple gameplay. The player (a strawberry) must move to catch cherries and increment his score.

The game can be played online at this link: http://files.myfrogbag.com/bli6du/FruitCatcher.swf

CONTROLS
--------
Arrow Keys:	Move
Z:		Jump

Moving while crouching allows the player to make a tall hop, which will get them over larger obstacles.

Built with Flixel 2.5.
